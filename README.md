## Marketing Analytics Dashboard (http://dashboard-nine-lake-57.vercel.app/leads)
A modern marketing analytics dashboard built using React, Tailwind CSS, and Recharts.
This project simulates a real-world admin panel with authentication, protected routing, API integration, and data visualization.

## Tech Stack
React (Vite)
React Router DOM
Tailwind CSS
Recharts (Data Visualization)
DummyJSON API (Dynamic Data)

## Features
Login Authentication (LocalStorage based)
Protected Routes
Interactive Analytics Charts
Lead Management Table
API Data Fetching
Modular Component Architecture

## Dashboard Includes
KPI Cards (Total Leads, Conversion Rate, etc.)
User Analytics Chart
Lead Management Section
Navigation with Logout functionality

## Installation
npm install
npm run dev
npm install  tailwind css

## Purpose

This project demonstrates:
Component-based architecture
State management with hooks
API integration using fetch
Conditional routing (ProtectedRoute)
Real-world dashboard UI design
